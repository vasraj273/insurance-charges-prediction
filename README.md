# Insurance Charges Prediction

This project analyzes insurance cost data and builds regression models to predict charges.

## Techniques Used
- Exploratory Data Analysis
- Linear Regression
- Random Forest Regressor
- Model Evaluation (R², MAE, RMSE)
- Log Transformation Experiment
- Streamlit Deployment

## Key Insights
- Smoking significantly increases insurance charges.
- Age and BMI positively influence cost.
- Random Forest outperformed Linear Regression.

## Deployment
Interactive Streamlit app included.
